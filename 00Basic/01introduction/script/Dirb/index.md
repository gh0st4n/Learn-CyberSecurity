# Script
[simpledirb.py](simpledirb.py)
# Usage

```
# Scan basic
python3 simpledirb.py http://target.com/

# Dengan wordlist kustom dan ekstensi
python3 simpledirb.py http://target.com/ /usr/share/wordlists/dirb/common.txt -X php,html,txt

# Dengan thread lebih banyak
python3 simpledirb.py http://target.com/ -t 20

# Simpan hasil
python3 simpledirb.py http://target.com/ -o hasil.txt
```
